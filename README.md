# 🎓 School Management Website

## 📌 Project Overview
This project is a web-based platform designed to manage and display school content in a structured and user-friendly way.

It focuses on organizing school data such as news, educational resources, and student information, with a simple content management system for teachers.

---

## ⚙️ Core Features

### 🏠 Home Page
- Displays latest school news (last 5 updates)
- Quick navigation links
- Clean and simple UI

### 📰 News System
- Teachers can:
  - Add news
  - Delete news
- Dynamic content display

### 👨‍🏫 Teacher Dashboard
- لكل معلم حساب خاص (Username & Password)
- صلاحيات:
  - إنشاء الأخبار
  - حذف الأخبار

### 📚 Educational Resources
- Organized resources for different grades
- Easy access structure

### 🏆 Top Students
- Display top students for each grade
- Structured data presentation

### 📞 Contact & Support
- External links integration (Facebook / LinkedIn)

---

## 🧠 Technical Highlights
- Built as a structured multi-page system
